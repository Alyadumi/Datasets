A simple TeX template to write responses in the peer-review process.

Commands \question and \answer allow for the quick generation of a response document.
Suitable for most scientific journals.
Leave a blank line before and after \question and \answer.
Do not pass parameters to these commands (e.g. "\question text" and not "\question{text}").

Script "gen_responses.sh" calls texi2pdf to generate the pdf.
It works on Mac OS X and Linux.
